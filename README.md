MetaROUTER-apps
===============

This is experimental "bare metal" embedded code that runs within the virtualized MIPS environment provided by Mikrotik's [RouterOS](http://www.mikrotik.com/software) [MetaROUTER](http://wiki.mikrotik.com/wiki/Manual:Metarouter).

## Requirements

[crosstool-ng](http://crosstool-ng.org/) is an easier way to build a cross-compiler and C libraries.  The .config configuration file needed for the MIPS MetaROUTER is provided in the subdirectory "crosstools-ng".
